2021/3/7 蓝桥杯校选

Posted on 2021-03-07

题目

A.风花雪月

奶牛发现了一款有趣的战旗游戏并沉迷其中。

不妨将战场视为一个二维平面，假设当前奶牛所在的位置为$(a,b)$，每次行动，他移动到的位置$(c,d)$必须满足与之前位置的曼哈顿距离小于等于$k$，即：

$ \left | a-c \right | + \left | b-d \right | \leq k$

幸运的是，不可思议的少女-苏谛斯，她会帮助奶牛，每次行动除了上述操作，他也可以选择：$(a,b)$瞬移到$(c,d)$，但需要满足以下条件中的任意一个：
$ a+b=c+d $
$ a-b=c-d $

现在奶牛希望从$(x,y)$移动到$(p,q)$并使行动次数最少，你能帮帮他吗？

Input

第一行输入一个数$k (1\leq k\leq 10^9)$

第二行输入两个数$x,y (1\leq x,y \leq 10^9)$，表示奶牛的初始位置

第三行输入两个数$p,q (1\leq p,q \leq 10^9)$，表示奶牛的最终位置

Output

输出一个数，表示最少的移动次数
